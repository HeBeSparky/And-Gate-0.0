# And-Gate-0.0
Demo of And Gate

Using a controller is advised.

Gamepad Controls
----------------------------
Left Joystick: move
RT: Doge roll
LT: Follow
A: Attack
X, Y, B: Ability buttons
Start: pause
Select: Full screen toggle

Demo Controls
----------------------------
4: Toggle camera
5: Spawn enemy
6: Cycle through enemies
7: Kill all enemies
8: Hitbox view
9: Debug mode toggle
0: Full screen toggle
